## Contributing

We welcome contributions from everyone! This is a "free for all" project where anyone can contribute improvements, bug fixes, or new features.

### How to Contribute

1. **Fork the repository** on GitHub
2. **Create a feature branch** (`git checkout -b feature/amazing-feature`)
3. **Make your changes** and commit them (`git commit -m 'Add amazing feature'`)
4. **Push to your branch** (`git push origin feature/amazing-feature`)
5. **Open a Pull Request** on GitHub

### Contribution Guidelines

- Keep code clean and well-commented
- Follow existing code style and conventions
- Test your changes with both small and large repositories
- Update documentation (README.md, AGENTS.md) if needed
- Be respectful and constructive in discussions

### What to Contribute

**Bug Fixes**: Found a bug? Please report it or submit a fix!

**Features**: Ideas for new features are welcome:
- New visualization modes
- Additional metrics and filters
- Performance improvements
- UI/UX enhancements
- Export capabilities
- Mobile responsiveness

**Documentation**: Help improve:
- User guides and tutorials
- Technical documentation
- Code comments
- Example use cases

**Testing**: Add unit tests, integration tests, or improve test coverage

**Performance**: Optimize database queries, rendering, or data processing

### Code of Conduct

Be kind, respectful, and inclusive. This is a community project for everyone.
